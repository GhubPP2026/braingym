# 🧠 BrainGym — Kids Cognitive Skills App

A fully offline-capable, single-page web application for kids aged 7–16 to develop focus, attention, creativity, cognitive skills, problem solving, and time management.

## 🌐 Live App

## ✨ Features

- **28 techniques** across 6 skill areas: Focus, Creativity, Problem Solving, Cognitive, Social & Emotional, Time Management
- **5 interactive games** playable directly in the browser:
  - 🎨 Stroop Effect Challenge
  - 🍅 Pomodoro Focus Timer
  - 📦 Eisenhower Task Priority Sorter
  - 🎯 Memory Palace Builder
  - 🌬 Mindfulness & Breathing (Humming Bee, Alternate Nostril, Alpha Wave Binaural Beats)
- **XP & Level system** — 6 levels from Brain Beginner to BrainGym Legend
- **10 Achievements** to unlock
- **Progress tracking** with skill mastery bars
- **Offline ready** — works without internet after first load
- **Zero dependencies** — pure HTML, CSS, JavaScript

## 🚀 Quick Start

Just open `index.html` in any browser — no install, no server needed.

braingym/

## 🛠 Deploy to GitHub Pages

1. Push this folder to a GitHub repository
2. Go to repo **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Save — your app is live at `https://USERNAME.github.io/braingym`

## 📱 Device Support

Works on all modern browsers — Chrome, Safari, Firefox, Edge — on desktop, tablet, and mobile.

## 🔒 Privacy

All progress data is stored locally in the browser (localStorage). No data is sent to any server.
